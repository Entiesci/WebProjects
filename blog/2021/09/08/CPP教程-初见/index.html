C++ 教程

hello world

你好，世界！

C++的世界,很大.

C++是C语言的继承，它既可以进行C语言的过程化程序设计，又可以进行以抽象数据类型为特点的基于对象的程序设计，还可以进行以继承和多态为特点的面向对象的程序设计。C++擅长面向对象程序设计的同时，还可以进行基于过程的程序设计，因而C++就适应的问题规模而论，大小由之。
C++不仅拥有计算机高效运行的实用性特征，同时还致力于提高大规模程序的编程质量与程序设计语言的问题描述能力。

C++是世界上最通用的编程语言之一,与HTML不同的是,C++是编程语言,而HTML是超文本标记语言.一个是编程,一个是标记,因此,C++的难度会大一些
与C++齐名的还要Java,Python等

C++的用处很大,在游戏搭建,软件构造等都有用武之地
C++是一门复杂的编程语言,同学们要做好心理准备

C++软件推荐:Dev C++

基本结构

接下来,我们正式开始学习C++这个编程语言

C++的基本代码框架如下

# include 
using namespace std;
int main()
{
程序
return 0;
}

C++的每个语句之间使用;隔开!
框架逐行讲解如下:

# include

这个是声明,<>里的是要使用的库,当你在使用C++里面的某些代码时,计算机可能需要在不同的库里面找到,而你必须声明所有你用的的库,不然就会报错没有这个代码

实例里的bits/stdc++.h是万能头文件,里面涵盖的大部分常用库

using namespace std;

这个的意思是——使用通用命名空间
这个是用来处理重名问题的,一般情况下这样写就可以

int main
{
...
return 0;
}

这个是主函数,表示程序主要执行部分,除了定义新函数需要写在外面&定义变量,列表可以写在外面 以外,其他的都要写在里面才会执行

return 0;

这个表示程序结束,终止.
